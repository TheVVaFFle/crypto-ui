# React Redux Startup App

### Overview
A basic crypto ui displaying a few of the top cryptos. Basic search function and ability to select different cryptos.

Webpack v3 
React v16  
Redux v3  

### Installation
Runs on http://localhost:3000/
```sh
$ cd crypto-ui
$ npm install 
$ npm start
```

### Production Build
Will output to app/
```sh
$ npm run build
```
